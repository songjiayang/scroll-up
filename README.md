# Scroll Page to Top

scroll-up is a simple Rails engine for scrolling page to top.

## Installation

Add this line to your application's Gemfile:

    gem 'scroll-up', '~> 0.0.5'

And then execute:

    $ bundle


## Usage

require js

```js
//= require scroll-up
```

require css

```scss
*= require scroll-up
```

if scss, just
```scss
@import "scroll-up";
```
